Document structure

Document structure

      
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width initial-scale=1.0">
  <title>Your Title</title>
</head>
  <body>
    Example text
  </body>
 </html>

- <!DOCTYPE html>: Declares the HTML version.

- HTML documents start with <!DOCTYPE html> to specify the document type.

- The entire document is enclosed within <html> tags.

- Use <head> to include metadata like character set and viewport settings.

- Title your page with <title> inside the to define what appears in the browser tab.

- The main content goes inside <body>.

- <body>: Contains the content of the HTML document.



Headings

Headings

      

<h1>This is Heading 1</h1>


<h2>This is Heading 2</h2>


<!-- ... -->


<h6>This is Heading 6</h6>

Use <h1> to <h6> for headings (h1 is the largest, h6 is the smallest).



Paragraphs

Paragraphs

      
<p>This is a paragraph of text.</p>

Define paragraphs with <p>



Links and lists

Links

      
<a href="https://www.example.com">Visit Example.com</a>

Create hyperlinks with <a>.



Lists

        
<ul>
  <li>Item 1</li>
  <li>Item 2</li>
</ul>

<ol>
  <li>Item 1</li>
  <li>Item 2</li>
</ol>

Use <ul> for unordered lists (bulleted) and <ol> for ordered lists (numbered).
List items are defined with <li>.



Images

Images

    
<img src="image.jpg" alt="Description of the image">

Use <img> to display an image.



Tables

Tables

      
<table>
    <tr>
        <th>Header 1</th>
        <th>Header 2</th>
    </tr>
    <tr>
        <td>Data 1</td>
        <td>Data 2</td>
    </tr>
</table>

Define tables with <table>, rows with <tr>, headers with <th>, and data cells with <td>.



Forms

Forms

      
<form>
    <label for="username">Username:</label>
    <input type="text" id="username" name="username">
    <button type="submit">Submit</button>
</form>

Use <form> to create a form, <input> to create an input fields, <label> to create labels, and <button> to create buttons



General Tips:

  • Always close your tags properly (e.g., <tag></tag>)
  • Use indentation for better readability.
  • Test your HTML in different browsers to ensure compatibility.
